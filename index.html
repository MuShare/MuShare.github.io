Rate Assistant

Rate Assistant is an exchange rate app, which provides real time rate, rate history and customer notification.

1. Search current and historical currency exchange rate.

2. Display favorite currencies on the home page after signing in.

3. Monitoring specific exchange rate by subscription.

4. Push notification to your iOS devices when the exchange rate reaches the threshold.

Httper

Httper is a REST API test tool running on your iOS devices. It helps developers to test their REST API anywhere and anytime without desktops.

Some features of Httper:

1. Request and show response with pretty JSON.

2. Project management

3. Some Internet tools like ping, whois and IP info.
